Good Dog Spa — ezyVet Integration Executive Summary

01 Executive Summary

A quick overview of what this integration means for your business

The Good Dog Spa website currently has a fully functional credit-based booking system that allows your customers to purchase credit packages, view their balance, and book daycare and hydrotherapy sessions. Right now, this system runs independently using demo data.

This proposal outlines how we can connect that system directly to your existing ezyVet software, so that credits, client details, pet information, and bookings all stay in sync — eliminating double handling and giving your customers a seamless self-service experience.

The goal: When a customer logs into the Good Dog Spa portal, they see their real credit balance from ezyVet, their real pets, and their real bookings — and can purchase credits and make bookings that automatically appear in ezyVet for your team.

Your ezyVet system already manages clients, animals, appointments, and wellness plan credits. The Good Dog Spa portal has matching features that can be linked directly:

03 How It Works

A simplified view of how the systems connect

ezyVet provides a secure API (a way for software systems to talk to each other) that allows authorised applications to read and write data. We build a secure middle layer that sits between the Good Dog Spa website and ezyVet, handling all the communication.

What happens when a customer uses the portal:

Viewing credits & bookings: The portal requests the customer's data from our secure server, which fetches it from ezyVet's API in real time. The customer sees their actual wellness plan credit balance and upcoming appointments.

Purchasing a credit package: The customer selects a package and pays via secure payment processing. Our server then creates the corresponding wellness plan membership or adds credits in ezyVet, so your team sees the purchase immediately.

Making a booking: The customer picks a date and session type. If they have available credits, the booking is created as an appointment in ezyVet and the credit is consumed — all automatically.

Good Dog Spa — ezyVet Integration Implementation Plan

05 Implementation Plan

The work broken into clear phases

PHASE 1 ~1 week

API Access & Configuration

  • Register a Private Integration with ezyVet (requires your approval as the clinic)
  • Obtain API credentials (partner ID, client ID, secret) for Wigram Vet's ezyVet instance
  • Map your existing ezyVet wellness plan products to the 5 Good Dog Spa credit types
  • Set up a secure development environment for testing
PHASE 2 ~2–3 weeks

Secure Backend Server

  • Build a secure server that sits between the website and ezyVet
  • Implement OAuth 2.0 authentication and automatic token management
  • Add rate limiting and caching to stay within ezyVet's API limits (60 requests/min)
  • Build customer authentication (matching portal logins to ezyVet contacts)
PHASE 3 ~2–3 weeks

Read Integration (Viewing Data)

  • Connect dog profiles — pull real pet data from ezyVet (breed, weight, health info)
  • Connect credit balances — show real wellness plan credit balances
  • Connect bookings — display actual upcoming and past appointments
  • Connect transaction history — show real purchase and usage records
PHASE 4 ~2–3 weeks

Write Integration (Creating Data)

  • Package purchases — create wellness plan memberships in ezyVet on purchase
  • Booking creation — create appointments in ezyVet from the portal
  • Credit consumption — automatically deduct billing credits when bookings are confirmed
  • Integrate real payment processing (Stripe or Windcave) linked to ezyVet invoicing
PHASE 5 ~1–2 weeks

Testing, Polish & Launch

  • End-to-end testing with real ezyVet data in a safe test environment
  • Staff training — how bookings and credits appear in ezyVet
  • Soft launch with a small group of customers
  • Full rollout with monitoring and support

Estimated total timeline: 8–12 weeks from API access approval to full launch, depending on ezyVet's onboarding speed and any configuration needed on the ezyVet side.
